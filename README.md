Worker Task Status Check
A simple JavaScript practice that loops through worker data and checks task statuses.
What it does
Iterates over an array of workers
Logs an error if any task is marked as donelate
Checks unfinished tasks using a while loop
Purpose
Practice arrays of objects
Use for and while loops
Work with conditional logic (if, logical OR)


(practice 2)

Action Movies Filter
A simple JavaScript exercise that loops through a list of movies and prints the first three movies with the Action genre.
What it does
Iterates over an array of movie objects
Checks the movie genre
Logs only the first 3 Action movies
Stops the loop using break
Purpose
Practice for...of loop
Use conditional logic
Control loop flow with a counter and break

(practice 3)

Object Destructuring
A simple JavaScript example showing how to extract properties from an object and reassign variables.
Key points:
Extract lastname and age from an object using destructuring
Reassign lastname without affecting age
Print values using console.log
Purpose: Practice object destructuring and variable assignment in JavaScript.
